域名免费拿！一分钟学会 .us.kg 域名申请并配置到 Cloudflare

Sun Xiaofei2024-11-052024-11-05

0 成本获取.us.kg域名：从注册到 Cloudflare 托管全攻略！

想要创建一个属于自己的域名并托管到全球网络节点加速的 Cloudflare 吗？本文将手把手教您如何免费领取一个 .us.kg 域名，并将其配置到 Cloudflare。内容详细、步骤清晰，即使是新手也可以轻松上手！

准备工作

在开始前，请确保您已具备以下条件：

  1. 一个有效的邮箱，便于注册和接收通知。
  2. 一个 Cloudflare 账户，如果还没有，请提前注册（下文会有详细步骤）。
  3. VPN 工具（可选），由于地区限制，部分用户可能需要 VPN 才能访问 .us.kg 域名注册网站。
  4. 身份信息生成 可以避免使用个人信息

如何注册 .us.kg 域名

1. 访问 .us.kg 域名注册网站

首先，访问 NIC.US.KG 网站.

2. 注册并申请新域名

  • 步骤 1：注册账户
    如果您还没有在 NIC.US.KG 注册账户，点击REGISTER A NEW DOMAIN按钮，然后使用身份信息生成按照提示填写基本信息

  • 步骤 2：申请域名
    确认域名可用后，返回主页并再次输入域名。在确认域名可用后，系统会提示您填写相关信息。填写完成后，提交注册请求。

  • 步骤 3：完成 KYC 验证
    选择 KYC 方式：GitHub KYC Verification 根据提示通过验证

  • 步骤 4：申请域名

    点击Domain name registration申请你的 us.kg 域名

    输入想要注册的域名，点击Check

    一般来说是秒通过

Cloudflare 账户创建与设置

1. 注册 Cloudflare 账户

如果您还没有 Cloudflare 账户，请前往 Cloudflare 官网 并注册新账户。Cloudflare 提供免费套餐，适合大部分个人网站使用。

  • 点击首页的“注册”按钮，输入您的邮箱和密码即可完成账户注册。
  • 完成注册后，Cloudflare 会要求验证您的邮箱，请前往您的邮箱并点击确认邮件中的链接完成验证。

2. 添加网站到 Cloudflare

  • 登录 Cloudflare 后，点击页面上的添加域。
  • 在弹出的窗口中，输入您刚刚注册的 .us.kg 域名（如 example.us.kg），点击继续。
  • 系统会自动扫描您域名的 DNS 记录，如果您是新域名，可能没有任何记录，点击“继续”跳过这一步。

将 .us.kg 域名托管到 Cloudflare

1. 获取 Cloudflare 的 DNS 服务器地址

  • 在 Cloudflare 的域名设置页面，您会看到 Cloudflare 提供的两组 DNS 服务器地址，例如：
    • ns1.cloudflare.com
    • ns2.cloudflare.com
  • 请复制这两个地址。

2. 返回 NIC.KG 设置 DNS

  • 登录您的 NIC.KG 账户，找到您刚刚注册的 .us.kg 域名管理页面。
  • 在 DNS 设置页面，替换默认的 DNS 地址为 Cloudflare 提供的地址。
    • ns1.cloudflare.com
    • ns2.cloudflare.com
  • 保存设置并退出。DNS 修改可能需要 24 小时生效，但通常只需几小时。

验证域名是否成功托管

返回 Cloudflare，点击域名的“检查”按钮，Cloudflare 会自动验证您的 DNS 配置是否正确。当系统检测到您的域名已正确指向 Cloudflare 的 DNS 服务器时，您的域名将显示为“已激活”状态。

总结

恭喜！到这里，您已经成功注册了一个免费的 .us.kg 域名，并将其托管到 Cloudflare。通过 Cloudflare 的全球节点加速和免费的 HTTPS 证书，您可以为用户提供更快速、安全的访问体验。
